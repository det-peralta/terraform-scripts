# Setup

#### 1. Create terraform.tfvars
```
mv terraform.tfvars.sample terraform.tfvars
```

#### 2. Edit variables in "terraform.tfvars", add proxmox auth data

#### 3. Add or create public key "terraform.pub"

#### 4. Run:
```
terraform init
```

#### 5. Run:
```
terraform plan
```

#### 6. Run:
```
terraform apply
```
